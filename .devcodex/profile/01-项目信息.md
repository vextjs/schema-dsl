## 技术栈

### 主语言
- **TypeScript**（源码位于 `src/`，构建产物位于 `dist/`）
- 构建工具：**tsup**（生成 ESM + CJS + 类型声明）

### 核心生产依赖

| 依赖 | 版本 | 用途 |
|------|------|------|
| `ajv` | `^8.17.1` | JSON Schema 验证引擎（Draft 7） |
| `ajv-formats` | `^2.1.1` | ajv 格式扩展（email、uri、date 等） |
| `cache-hub` | `^1.0.0` | LRU 缓存后端（`MemoryCache`，`CacheManager.ts` 使用） |

> ⚠️ 仅 3 个生产依赖，保持极低依赖树。

## npm 导出配置

```json
{
  "type": "module",
  "main": "./dist/index.cjs",
  "exports": {
    ".": {
      "import": {
        "types": "./dist/index.d.ts",
        "default": "./dist/index.js"
      },
      "require": {
        "types": "./dist/index.d.cts",
        "default": "./dist/index.cjs"
      }
    }
  }
}
```

> `main` 字段指向 `dist/index.cjs`，确保 `require('schema-dsl')` 在纯 CJS 环境中仍可工作。
