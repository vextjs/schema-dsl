# schema-dsl v2.1.3

Released: 2026-07-08

## Release theme

This patch closes the Low findings from the 2026-07-08 project audit. It adds a clearer security note for executable locale files, reduces transient host-load noise in performance smoke tests, and adds a no-dependency documentation link checker to the release gate.

## Changed

- Added a bilingual security checklist note that `.js` and `.cjs` locale files should be treated as trusted application code.
- Documented `codeLocaleFiles: 'deny'` as the safer policy for data-only or untrusted locale directories.
- Made performance smoke tests use a small best-of sample window so a single scheduler pause does not fail the gate.
- Added `npm run docs:linkcheck` to validate README / docs Markdown relative links and Rspress sidebar/nav local targets.
- Included `docs:linkcheck` in `npm run test:all:with-audit` and documented the command in the README development script list.
